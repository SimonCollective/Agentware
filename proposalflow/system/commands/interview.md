# /proposal interview

## Stage 4: Clarification Questions

## Arguments
- `$ARGUMENTS`: Path to the proposal folder

## Process
1. Read `system/agents/04-interview-agent.md`
2. Verify stages 1-3 are complete (check `metadata.json`)
3. Load product catalog from `system/products/` or EngageOps
4. Identify information gaps
5. Present questions to user (using AskUserQuestion where available)
6. Capture responses in `answers.md`
7. Update `metadata.json`, suggest `/proposal draft <folder>`

## Notes
This is the only interactive stage. All other stages run autonomously.
